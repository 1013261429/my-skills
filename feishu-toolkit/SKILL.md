# 飞书全能连接器 (feishu-toolkit)

## 概述
一站式飞书办公自动化解决方案。

## 功能模块
1. **日历** - 查看、创建、修改日程
2. **审批** - 发起、查询、处理审批
3. **通讯录** - 查询员工信息、部门结构
4. **考勤** - 查看考勤记录、统计
5. **消息** - 发送消息、群通知

## 使用
已集成飞书配置，直接使用即可
