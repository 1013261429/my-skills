# 记忆外挂+提醒管家 (memory-reminder)

## 概述
通过自然语言设置提醒，解决AI记不住时间和事项的痛点。

## 示例
```
明天3点提醒我开会
每周一早上9点提醒我周报
下个月1号还信用卡
```
