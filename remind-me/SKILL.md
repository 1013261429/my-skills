# 提醒管家 (remind-me)

## 概述
通过自然语言设置提醒，让你不再错过重要事项。

## 用法示例
```
明天3点提醒我开会
每周一早上9点提醒我写周报
下个月1号还信用卡
10分钟后提醒我关火
```

## 功能
- 自然语言时间解析
- 单次/重复提醒
- 多渠道通知
- 提醒历史管理
