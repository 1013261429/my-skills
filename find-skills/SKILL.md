# 技能雷达 (find-skills)

## 概述
智能发现和推荐最适合你需求的OpenClaw技能。

## 功能
1. **需求分析** - 理解你的具体需求
2. **技能匹配** - 从ClawHub搜索匹配的技能
3. **智能推荐** - 基于评分和下载量推荐
4. **一键安装** - 简化安装流程

## 用法
```
帮我找一个能处理PDF的技能
推荐几个好用的搜索技能
查看本周热门技能
```
